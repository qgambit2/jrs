# JRS — Java Review Swarm

Landing page for **JRS (Java Review Swarm)** — multi-perspective Java code
review with adversarial verification, as a Claude Code plugin.

- **Live site:** https://qgambit2.github.io/jrs/
- **Field-test results:** https://qgambit2.github.io/java-review-results/

A static single-page site (`index.html`), served via GitHub Pages. No build step.
